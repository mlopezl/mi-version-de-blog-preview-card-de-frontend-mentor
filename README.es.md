## Descripción General
Este proyecto es una **tarjeta de vista previa de blog responsive** que se adapta a diferentes tamaños de pantalla.  
Incluye una imagen del blog, una etiqueta de categoría “Learning”, un título, una breve descripción y un pie de tarjeta con el avatar del autor, su nombre y la fecha de publicación.

---

## El Desafío
Los usuarios deberían poder:

- Ver el diseño de la tarjeta según el tamaño de pantalla de su dispositivo.  
- Observar efecto hover en el título para mayor interactividad.  
- Leer claramente la información del blog gracias a un marcado semántico y bien estructurado.

## Mi Proceso
1. Estructuré el HTML usando elementos semánticos (`<article>`, `<header>`, `<footer>`).  
2. Estilicé el layout con CSS y usé **Flexbox** para la alineación.  
3. Implementé efecto hover sobre el título para que sea interactivo.  
4. Aseguré que el componente sea totalmente responsive ajustando tamaños, márgenes y escala tipográfica.

---

## Tecnologías Usadas
- HTML5  
- CSS3  
- Flexbox

## Lo Que Aprendí
- Cómo convertir un diseño en HTML + CSS real.  
- Cómo aplicar efecto hover para mejorar la experiencia del usuario.  
- Cómo gestionar tipografía, espaciado y alineación en un componente compacto.  
- La importancia del HTML semántico para la estructura y accesibilidad.
